# FRC Team 5193 — Robot Code
**Saginaw Arts and Sciences Academy | Saginaw, MI**

*Robot code developed collaboratively by the FRC Team 5193 software subteam.*

Java-based robot code for FRC Team 5193 across two competition seasons, built using WPILib.

---

## 2023 — Charged Up

In Charged Up, robots competed to pick up and place cone and cube game pieces onto tiered nodes in a grid structure.

**Key Features:**
- Swerve drive implementation for omnidirectional movement and precise positioning
- Autonomous routines for game piece placement without driver input
- Teleoperated control for picking up and placing game pieces onto elevated platforms and poles

**Built with:** WPILib, Java, REV Robotics

---

## 2022 — Rapid React

In Rapid React, robots competed to pick up and shoot cargo (balls) into elevated hub targets.

**Key Features:**
- Ball intake and shooting mechanism targeting elevated hoops
- Autonomous shooting and drive routines
- Teleoperated drive and shooter control

**Built with:** WPILib, Java, REV Robotics

---

## About FRC

FIRST Robotics Competition (FRC) is an international high school robotics competition where teams of students design, build, and program robots in six weeks to compete in a game announced each January.
