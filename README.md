FRC Team 5193 — Robot Code
Java-based robot code for FRC Team 5193 (Saginaw Arts and Sciences Academy) across two competition seasons.
2023 — Charged Up
Robot capable of picking up and placing game pieces onto elevated platforms and poles using autonomous and teleoperated control.
2022 — Rapid React
Robot capable of picking up and shooting balls into elevated hoops, with autonomous targeting and teleoperated drive control.
Built with: WPILib, Java, REV Robotics, CTRE Phoenix
